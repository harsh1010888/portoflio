### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor for customization (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/harsh1010888/portoflio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portoflio
   ```

3. Open `index.html` in your web browser:
   - Double-click the `index.html` file, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local server like `python -m http.server` or `npx serve`

## ğŸ¨ Customization

### Updating Content

1. **Personal Information**: Edit the text content in `index.html`
   - Update the hero section with your name/tagline
   - Modify the About Me section with your bio
   - Add your actual projects in the Projects section
   - Update the contact email in the Contact section

2. **Styling**: Customize colors and fonts in `styles.css`
   - Change the color scheme by modifying the color values
   - Adjust fonts by updating the `font-family` properties
   - Modify spacing and layout as needed

3. **Adding Projects**: In the `index.html` file, duplicate the project card structure:
   ```html
   <div class="project-card">
       <h3>Your Project Name</h3>
       <p>Project description here.</p>
   </div>
   ```
